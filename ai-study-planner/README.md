# 📚 AI Study Planner using Endee (Vector Concept)

## 🚀 Project Overview
This project is a simple AI-based Study Planner that suggests study tasks based on user input. It demonstrates the concept of semantic search using a vector-like approach.

## 🎯 Features
- Generate study plan
- Search based recommendations
- Simple AI logic
- Beginner-friendly implementation

## 🧠 Use of Endee
Endee is a vector database used for semantic search applications.

In this project:
- Study data is treated as vector-like information
- User query is matched with stored data
- Similar results are returned (semantic search concept)
